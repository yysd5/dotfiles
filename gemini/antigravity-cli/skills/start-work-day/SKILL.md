# 1日の業務を開始する

1日の業務を開始する。

具体的には、以下を実施する

1. `~/work-repo/work-log`配下に、本日用のディレクトリを作成

形式は`./2026/0305/`のようにする

2. 作成したディレクトリに本日用のメモを書くmdを配置する(`{yyyymmdd}-worklog.md`)

テンプレートは以下
```
# {yyyymmdd}-worklog

## やったことについてのメモ

## 本日のサマリ

### 実施作業と工数

### 想定される残作業

### まとめ

```
