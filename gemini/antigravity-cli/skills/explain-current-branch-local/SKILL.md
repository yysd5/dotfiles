---
name: explain-current-branch-local
description: ローカルの現在のブランチと、引数(default=develop)に指定されたブランチの差分に対して解説を行う。
---

以下の手順で、ローカルの現在のブランチと{{args}}ブランチの差分に対して解説を行ってください。

1. 現在のブランチの確認:

`git branch --show-current` を実行して、現在チェックアウトしているブランチを確認してください。

2. 差分の取得:

`git diff {{args}}...HEAD` コマンドを使用して、{{args}}ブランチから現在のブランチへの差分（コミット済み）を取得してください。
未コミットの変更やUntrackedなファイルは対象外とします。

なお引数の指定が無ければ、developブランチとの差分を取得で良いです。

3. 解説の実施:

取得した差分に基づき、解説を行ってください。
- 推測される背景や目的
- 目的を達成するためにどんな修正が行われているか
などの観点を中心にお願いします。
