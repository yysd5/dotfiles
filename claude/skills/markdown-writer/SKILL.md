# Markdown作成ガイドライン

(linter使っていない場合以下を守る)

## 全般的なルール

- 箇条書きを中心に構造化する
- インデントはwhitespace 4つとする

## 改行に関するルール

- 基本的に1行1文。異なる文を書く時は特別な理由がない限り改行する
- 横に長過ぎる文章は避ける。大体1行英数字換算で100文字くらいを目安とする。
  日本語の場合は50-60文字くらい。同じ文内でも長ければ改行する
- 見出しと本文の間は常に1行改行を入れること
- 見出しと見出しの間は常に1行改行を入れる
